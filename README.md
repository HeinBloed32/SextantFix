# SextantFix


SextantFix
Celestial navigation app for sailing — Tagbogenverfahren & St.-Hilaire sight reduction.
🇩🇪 Deutsch · 🇬🇧 English · 🇫🇷 Français
---
Deutsch
SextantFix ist ein Werkzeug zur astronomischen Navigation (Tagbogenverfahren und St.-Hilaire-Verfahren) für echtes und simuliertes Segeln.
Installation (Windows)
Unter Releases → Latest die Datei `SextantFix\_Setup.bat` herunterladen.
Die heruntergeladene Datei starten. Bei einer Windows-Sicherheitsmeldung auf „Weitere Informationen" → „Trotzdem ausführen" klicken.
Das Setup lädt eine eigene Python-Umgebung, installiert die Abhängigkeiten und baut das Programm lokal. Das dauert einige Minuten. Am Ende liegt eine Verknüpfung auf dem Desktop.
Eine ausführlichere Anleitung liegt dem heruntergeladenen Paket bei.
macOS
Für macOS gibt es derzeit keine offiziell unterstützte Version, da sie nicht regelmäßig getestet werden kann. Auf Apple-Silicon-Macs (M1/M2/M3/M4) ist eine `.dmg` auf Anfrage erhältlich. Intel-Macs werden nicht unterstützt.

English
SextantFix is a tool for celestial navigation (Tagbogen method and St. Hilaire intercept method) for real and simulated sailing.
Installation (Windows)
From Releases → Latest, download `SextantFix\_Setup.bat`.
Run the downloaded file. If Windows shows a security prompt, click "More info" → "Run anyway".
The setup downloads its own Python environment, installs the dependencies and builds the application locally. This takes a few minutes. A desktop shortcut is created at the end.
A more detailed guide is included in the downloaded package.
macOS
There is currently no officially supported macOS version, as it cannot be tested regularly. A `.dmg` is available on request for Apple Silicon Macs (M1/M2/M3/M4). Intel Macs are not supported.

Français
SextantFix est un outil de navigation astronomique (méthode Tagbogen et méthode de la droite de hauteur de St. Hilaire) pour la voile réelle et simulée.
Installation (Windows)
Depuis Releases → Latest, téléchargez `SextantFix\_Setup.bat`.
Lancez le fichier téléchargé. Si Windows affiche un avertissement de sécurité, cliquez sur « Informations complémentaires » → « Exécuter quand même ».
Le programme d'installation télécharge son propre environnement Python, installe les dépendances et compile l'application localement. Cela prend quelques minutes. Un raccourci est créé sur le bureau à la fin.
Un guide plus détaillé est inclus dans le paquet téléchargé.
macOS
Il n'existe actuellement aucune version macOS officiellement prise en charge, car elle ne peut pas être testée régulièrement. Un fichier `.dmg` est disponible sur demande pour les Mac Apple Silicon (M1/M2/M3/M4). Les Mac Intel ne sont pas pris en charge.
